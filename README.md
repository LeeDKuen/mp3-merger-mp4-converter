# MP3 Merger + MP4 Converter

Utilities for:
- merging multiple MP3 files into one file
- converting MP3 files to MP4 with a still background image

## Project Structure

- `mp3_merger/app.py`
- `mp4_converter/app.py`

## Requirements

- Python 3.9+
- `pydub`
- `Pillow`
- `ffmpeg` available in `PATH` (for MP4 conversion)

## Install

```bash
pip install pydub Pillow
```

## Run

```bash
python mp3_merger/app.py
python mp4_converter/app.py
```
